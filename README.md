# GigShield AI 🛡️

AI-powered parametric insurance platform for gig delivery workers in India.

## Quick Start

### 1. Start Backend
```bash
cd backend
npm install
npm start
```
Backend runs on: http://localhost:5000

### 2. Start Frontend
```bash
cd frontend
npm install
npm start
```
Frontend runs on: http://localhost:3000

## Features
- User Onboarding with city & platform selection
- AI Risk Assessment (weather + AQI analysis)
- Parametric Policy Generation (weekly, auto-renew)
- Simulate Disruptions (Rain / AQI / Curfew)
- Automatic Claim & Payout System
- Fraud Detection (Low/Medium/High)
- Dashboard with Charts

## Demo Flow
1. Fill onboarding form → 2. View AI risk score → 3. Activate policy → 4. Simulate disruption → 5. Watch auto-payout
